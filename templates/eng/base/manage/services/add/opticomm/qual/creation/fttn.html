Voiceband Continuity:


IMPORTANT: Voiceband continuity enables your existing voice services to continue to be provided over exchanged fed copper for up to 18 months from NBN Ready For Service date, after which time all PSTN pass through must migrate to a VOIP based service. The requirement for Transition Voiceband Continuity should be discussed with the End User prior to submitting the order. Failure to order Voiceband Continuity will result in the End User being without a Voiceband service (i.e. unjumper their existing telstra based phone line).
Splitter Installation:


A professional central splitter installation option is available during ordering as part of the initial order or as a subsequent modification order. Splitter installation will be provided by NBN Co. Different charges apply for each scenario and you should consult your price book. The central splitter is hardwired at the first Telecommunications Outlet (TO) in the end user premises.
